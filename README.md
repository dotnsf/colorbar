# ColorBar


## Overview

0x000 ～ 0xfff までのカラーバーを表示する。

実際には 0x111 から 2 ビットずつずらして 0xfff までの 2048 色をカラーバーにして Canvas 内に表示する。


```
   0x111
-> 0x11f
-> 0x1ff
-> 0x1f1
-> 0xff1
-> 0xf11
-> 0xf1f
-> 0xfff
```
